Return on Investment Calculator (in development)

This web app, which can be found at https://cafarooo.com/RoIcalculator, is a return on investment calculator which takes three forms of input and returns the value minus the cost. The goal for this project is to create a mobile app, so Jquery was used to make an intuitive and practical UI. Some notable features include:

  • Buttons that allow the viewport to travel a responsive device width in either direction
  • RNG - produced background color
  • local data storage
